# Sales Analysis Project (Python)

## 📊 Overview
This project analyzes simple sales data using Python and pandas.  
The goal was to practice data analysis skills and create a simple report.

---

## 📁 Dataset
The dataset contains sales information:
- date
- product
- category
- price
- quantity
- country

---

## 🧠 Key Analysis
In this project I:
- calculated revenue (price × quantity)
- analyzed top-selling products
- analyzed revenue by country
- created a simple bar chart visualization

---

## 📈 Key Insights
- Laptop is the highest revenue product
- Germany generated the highest total revenue
- Electronics is the strongest category

---

## 🛠 Tools Used
- Python
- pandas
- matplotlib
- Git & GitHub

---

## 🚀 Goal
To practice real-world data analysis workflow and prepare for junior data analyst role.
